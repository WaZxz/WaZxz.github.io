你好，Hexo

1.标识符要求：

  • 标识符由字母、数字、_、$所组成，其中不能以数字开头，不能用Java中的保留字（关键字）

  • 阿里编码规范要求：
    a.代码中的命名均不能以下划线或美元符号结束，也不能以其开始
    b.代码严禁使用拼音与英文，不允许使用中文

    2.类名
    类名是以大写字母开头的单词，如果类名由多个单词构成，则每个单词首字母的大写
    *eg. class Demo eg. class FirsrDem*

    3.变量和函数名称
    如果只含有一个单词则全部小写，如果变量名由多个字母组成，则从第二个字母开始，每个单词首字母大写。
    eg. int value
    eg.class firstValue

    3.常量的命名
    单词全部大写，多个单词用_隔开。
    eg. final int MY_INT_VALUE=10
